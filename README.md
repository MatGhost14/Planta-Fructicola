# Proyecto Completo: Inspección de Contenedores

## 🧩 Cómo ejecutar
1️⃣ Instalar dependencias
```bash
npm install
```
2️⃣ Ejecutar ambos servicios
```bash
npm run dev
```

- Backend: http://localhost:4000
